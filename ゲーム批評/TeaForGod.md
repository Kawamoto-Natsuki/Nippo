# ゲームタイトル: Tea For God
### 開発元: [Example Studio]
### 使用プラットフォーム:Oculus Quest 3
#### ジャンル: アクション、ローグライト、探索

## ゲーム概要
「ゴッドエンペラーは、私たちの世界を超えた場所に保持されています。ロボットの大群と戦いながら、巨大で奇妙な複合施設を探索してください。部屋の中を歩き回っている間ずっと、没入感を別のレベルに引き上げます。体験をカスタマイズして、強烈なシューター、ローグライト エクスプローラー、リラックスできるトレッキングを体験できます。
グラフィックスとビジュアル
グラフィックスは非常にリアルで、美しいファンタジー世界が広がります。特にVRならではの没入感が高く、まるで現実のような体験ができます。」



## ゲームプレイと操作性
操作性は直感的だったと思う。静止モードではコントローラー操作で酔いやすい点と床からモノを拾い上げるアクションが適さない印象だった。<br>
しかし、そもそもこのゲームが歩行モードで本領発揮するから度外視でよさそう。<br><br>
本題ともなる歩行モードは、実空間で2,3畳あれば満足に歩いた感覚になれる画期的な体験だった。
### 体験エリア
まず、ゲームスタートとともにQuest上でスキャンされたプレイエリアからゲームの体験エリアが決められる。

<img src = "https://github.com/Nats360/Nippo/assets/86301377/e6e591a9-84ed-48ff-bcc6-86813aed7c5e" width = 30%>

このエリア自体は物理的に狭いが、ゲーム中には狭さを感じることはない。理由は、このゲームが「手続き型生成」という**ローグライク**なステージ生成を採用しているからだ。ストーリーに基づいたシーンは存在するものの、そこにたどり着くルートは毎回異なる。<br>
そのため、同じ場所を何度も踏みしめることはなく、実際には同じ経路を歩いているのに、その実感はほとんどない。狭い範囲を広大に感じさせる要因はステージ生成だけではない。<br>
ステージ生成時には、新しい経路が生成される際に古い経路と重なることがあるが、興味深いことに、これらの道は互いに干渉しないようになっている。

Shaderについては詳しく触れないが、体験上、認知の不一致を感じることはあっても、それは不快感ではなく、面白い体験として受け取ることができるため、マイナスにはならない。また、次々に現れる敵を処理しなければならないため、「道が重なっていたのでは？」と感じる暇もなく、前進し続けることで歩き続けている感覚を維持できる。

<img src = "https://github.com/Nats360/Nippo/assets/86301377/a68ab337-02fb-413f-8a58-9e8c583b67bf" width = 30%>

さらに、世界観が荒廃した未来像を描いているため、艦内で不思議なことが起きても違和感なく進めることができ、**ストーリーや世界観**とのマッチ度も高く感じる。

そういわれてみれば「ローグライク×VR」には世界観のマッチ度って構成要素も念頭に置くべきかもしれない。
###　カスタム性
歩行体験に加えて、**武器選択の多様さ**も魅力の一つ。
敵を倒すことで武器をドロップしてくれる。またフェーズごとにアビリティみたいな機能を身に着けることで自分をグレードアップする楽しみも与えてくれえる。

これら機能を実装してるあたり、かなりアクションゲームとして楽しさを生んでるが、一つ言うならグレードアップした実感が得られないような気がする。

アビリティを身に着けることで脳力アップするが見た目が変わるわけでもなく、敵への影響が目に見えて分かるわけでもないので能力アップする楽しみが得ずらい印象。簡単に一つ改善案として、能力アップしたら敵討伐時に出るエフェクトが強くなる、少し派手になるだけでもいいし、自分の腕の形状変化でも何かしら簡単に視覚的なフィードバックを与えたい。
## サウンドと音楽
ディストピアにちなんだBGMのマッチ度は高く感じた。

SEはディストピアにしては軽さを感じたが、グラフィックスとのバランスを考えた結果だと感じる。

## リプレイ性
VR好きのプレイヤーからは空間のゆがみを体感できる画期的な体験を評価されてリピートできそう。

ただ、より一般曹によりそうならストーリーを終えてからの体験はリピートしずらいと感じる。
探索という割に秘宝を探すようなロマンがあるわけでもなく、自分を無制限に強くしてヌルゲー化させるような悪だくみをできるわけでもなく、着実にストーリーを進めていくしかない印象。

なのでリプレイ性を高めることが難しいなら一プレイの時間を延ばすことを考えてもいいかもしれない。
たとえば、謎解きの難易度をあげる、敵の難易度を上げる、謎解きのヒントは前ルートに存在するとか探索の必要性をあげる要素が必要かもしれない。
